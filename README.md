# ESP32 S3 Box player
It is suited for the first touchscreen version of the S3 box.
It is a sendspin media player which integrages with music assistant via home assistatant. See my music assistant package in home assistant for that.
It contains a basic player (play/pause, prev, next mute/unmute) with albumart and title and artist.
It also contains a favorite page with your favorite radio stations (marked as favorite in music assistant) there you can select a radio station to listen to
And a basic status page showing some stats. You can toggle between pages with the home button.
