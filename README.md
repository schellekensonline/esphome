# ESP32 S3 Box player with external PCM5102 audio output connected to dock
It is suited for the first touchscreen version of the S3 box with a dock icw. a PCM5102 audio DAC.
It is a sendspin media player that integrages with music assistant via home assistatant. 
See my music assistant package in my home assistant repo for that.

It contains a basic player (play/pause, prev, next mute/unmute) with albumart and title and artist.
It also contains a favorite page with your favorite radio stations (marked as favorite in music assistant) there you can select a radio station to listen to
And a basic status page showing some stats. You can toggle between pages with the home button.
It is work in progress. I will probably add some more functions later on
